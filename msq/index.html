My Current Project

Ali Kazerani

I am currently working on a problem concerning magic squares. I may provide details once this little project concludes, but for now, please allow me to be a little vague:

  • I am searching for magic squares of a particular form.
  • The objects I am searching for are not even guaranteed to exist.
  • The search is computationally intensive.
  • I used SymPy with Python to help perform an initial algebraic reduction, to find useful patterns in sets of algebraic expressions as well as in numerical sequences, and to symbolically and numerically verify elements of the C++ code discussed below.
  • I have written (and continue to write, improve, test, and run) C++ programs to carry out the steps of the core numerical search.
    • I am currently executing one stage of the search over a freshly expanded space. This run should take twenty days, after which I will be able to run the downstream stages.
    • For some tasks, I have implemented a couple of alternatives (e.g., memory-centered vs disk-centered, and using alternative approaches for concurrency).
    • From the standard library, I have been using containers, algorithms, smart pointers, futures (via async calls), threads, mutexes, condition variables, clocks, file streams, filesystem utilities, mathematical functions, etc.
  • I am using Git for version control.

(I last updated this page on 1 September 2022.)
